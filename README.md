# Pocket Bloom

Pocket Bloom is an original browser-based virtual-pet game inspired by the joyful handheld toys of the 1990s.

## Features

- Egg warming and animated hatch sequence
- Persistent care statistics stored in local browser storage
- Feeding, water, play, cleaning, sleep, medicine, exploration, and affection actions
- Growth stages and care-influenced evolution hints
- Multiple in-game environments
- Catch-the-starlight mini-game
- Achievements, streaks, generated sound effects, and offline stat decay
- Responsive mobile-first design
- Installable Progressive Web App

## Run locally

Open `index.html` directly, or serve the folder with any static server.

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.
